# Project Structure Guide

This repository works best when visitors can quickly find a project, understand what it is, and download the right files.

## Suggested top-level categories

```text
Conversions/
macros/
Mounts/
Toolheads/
Cable-Management/
Electronics/
Jigs/
Accessories/
```

You do not need all of these immediately. Add categories only when the collection grows enough to need them.

## Suggested project layout

```text
Category/Project-Name/
  README.md
  images/
  cad/
  stl/
  step/
  3mf/
  configs/
```

## README checklist

A good project README should answer:

- What printer is this for?
- What problem does it solve?
- What files are included?
- What hardware is needed?
- How should it be installed or used?
- Are there fitment limits or known issues?
