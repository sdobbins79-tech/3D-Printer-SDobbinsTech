# Contributing

Thanks for helping improve this repository.

This repo is meant to be a clean, download-friendly collection of 3D printer upgrades, conversions, and useful printer files.

## What belongs here

- Printable upgrade parts
- Hardware conversions
- Printer-specific mounts, ducts, brackets, and adapters
- Macros and config helpers
- CAD source files when available

## What to include with each project

Please give each project its own folder and include:

- A short `README.md`
- The printer model or platform
- A short description of what the project does
- The included file types such as `STL`, `STEP`, `3MF`, or macro files
- Basic install, print, or setup notes
- Any required hardware, screws, magnets, inserts, or wiring
- Known compatibility limits or fitment notes

## Suggested layout

```text
Category/Project-Name/
  README.md
  images/
  cad/
  stl/
  step/
  3mf/
  configs/
```

Only create the folders you actually need.

## Naming tips

- Use clear project names
- Mention the printer model when helpful
- Avoid vague names like `test` or `new part`
- Keep old revisions separated from the current recommended files

## Before uploading

- Make sure files open correctly
- Remove duplicates and failed exports
- Include enough notes for someone else to use the files
- Credit any original design you remixed or adapted
