# Hardware

Hardware upgrades and printed parts for supported printers.

This section is for printer-specific mechanical or hardware changes such as:

- Toolhead conversions
- Mainboard or electronics mounts
- Ducts, brackets, and structural add-ons
- Adapter parts for other hardware upgrades

## Current projects

### [FLSUN S1](./FLSUN%20S1)
Effector, toolhead, and other hardware upgrade parts for the FLSUN S1.

### [PID Chamber Heater Controller](./PID%20Chamber%20Heater%20Controller)
Standalone PID controller enclosure for a chamber heater setup. The PTC heater and fan assembly is still in progress.

## What each hardware folder should include

- A short `README.md`
- Compatible printer model
- Notes about required hardware
- Print or install notes when needed
- The actual project files grouped clearly
